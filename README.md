🚗 VINTAGE RIDE | Car Rental Platform

🌐 Live Website: Click Here to View Live Site

A modern and secure car rental platform where users can explore, list, and book cars with a smooth and user-friendly experience.


---

✨ Key Features

🔐 Secure Authentication
Users can sign up and log in using Email/Password or Google Authentication with secure JWT verification.

🚘 Dynamic Car Browsing
Explore available cars in a responsive grid layout with detailed information and smooth navigation.

🛠️ Complete CRUD Functionality
Logged-in users can add, update, and delete their own car listings easily.

📅 Easy Booking System
Book cars instantly with additional options like driver requests and special notes.

📂 Private Dashboard Access
Protected routes for managing My Bookings, My Added Cars, and Add Car pages.

💳 Booking History & Price Tracking
Users can view booking dates, total costs, and booking history in one place.

⚡ Responsive & Clean UI
Fully responsive design optimized for mobile, tablet, and desktop devices.
